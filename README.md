# Monte-Carlo-CFAR-for-Radar-Target-Detection
This project implements a Monte Carlo-based CFAR (MC-CFAR) algorithm for radar target detection. It uses random sampling of the Range-Doppler Matrix to estimate noise and apply adaptive thresholding, achieving efficient detection with controlled false alarm rates compared to traditional methods.
